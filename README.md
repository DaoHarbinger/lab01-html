## Опис проєкту

Цей проєкт є продовженням попередніх робіт (HTML + CSS). До вже існуючого тристорінкового сайту додано **10 обов’язкових JavaScript-механік**, які реалізують реальну клієнтську логіку:

1. **Підключення JS та структура коду** – єдиний файл `main.js` з функціями ініціалізації.
2. **Підсвічування активної сторінки** – автоматичне додавання класу `.active` до пункту меню.
3. **Бургер-меню** – кнопка з’являється на мобільних екранах (≤768px), відкриває/закриває навігацію.
4. **Перемикач світлої/темної теми** – змінює кольорову схему сайту, зберігає вибір у `localStorage`.
5. **Кнопка «Вгору» та поточний рік** – кнопка з'являється після прокрутки, плавно повертає наверх; у футері автоматично підставляється поточний рік.
6. **Акордеон (FAQ)** – блок запитань, які розгортаються/згортаються при кліку.
7. **Фільтрація карток** – кнопки для відображення всіх / тільки Front-end / тільки інструментів.
8. **Модальне вікно (lightbox)** – збільшення зображення по кліку; закриття по кнопці, overlay або Escape.
9. **Покращення форми** – валідація полів (ім’я, email, довжина повідомлення), лічильник символів для `textarea`, повідомлення про помилки.
10. **Чернетка форми та обробка submit** – дані форми автоматично зберігаються в `localStorage` і відновлюються після перезавантаження сторінки; при сабміті дані збираються через `FormData` і показується підтвердження (реальної відправки немає).

## Як запустити

1. Завантажте всі файли, зберігаючи структуру папок.
2. Відкрийте файл `index.html` у сучасному браузері (Chrome, Firefox, Edge).
3. Для перевірки мобільної версії:
   - Зменшіть ширину вікна браузера до <768px або використайте інструменти розробника (F12 → Device Toolbar).
4. Протестуйте всі інтерактивні елементи:
   - Кнопка теми (зміна кольорів, збереження після перезавантаження).
   - Кнопка «Вгору» (з'являється при скролі).
   - Акордеон (клік на питання).
   - Фільтри карток.
   - Модальне вікно (клік на зображення).
   - Форма: введіть дані, перезавантажте сторінку – вони відновляться; натисніть «Надіслати» – побачите повідомлення.
