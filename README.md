## Опис проєкту

Цей проєкт містить три HTML-сторінки, які демонструють:

- Семантичну структуру HTML5 (header, nav, main, footer)
- Роботу з текстом (заголовки, параграфи, strong, em)
- Списки (марковані та нумеровані)
- Таблиці (caption, thead, tbody)
- Зображення з підписами (figure, figcaption)
- Форми з різними типами полів, радіокнопками, чекбоксами та HTML5-валідацією
- Вбудоване відео та аудіо з альтернативними джерелами
- Бонусне завдання – кнопка з динамічною зміною тексту (JavaScript)

## Як запустити

1. Завантажте всі файли в одну папку, зберігаючи структуру.
2. Відкрийте `index.html` у будь-якому сучасному браузері (Chrome, Firefox, Edge).
3. Використовуйте навігацію для переходу між сторінками.
4. Для роботи форми: заповніть поля, натисніть "Надіслати" – побачите дані в адресному рядку (метод GET).
5. Відео та аудіо вимагають інтернет-з'єднання, оскільки використовують зовнішні URL.
