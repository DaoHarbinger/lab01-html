## Опис проєкту

Цей проєкт є виконанням практичних робіт №1 (основи HTML) та №2 (основи CSS). Розроблено тристорінковий веб-сайт з єдиним дизайном, адаптивністю та інтерактивними елементами.

### Сторінки:
- **Головна (`index.html`)** – текст, списки, таблиця, зображення, блок навичок (CSS Grid).
- **Про мене (`pages/about.html`)** – відео та аудіо з підписами, блок інформації, Flexbox-композиція.
- **Контакти (`pages/contact.html`)** – форма зворотного зв'язку з різними типами полів, валідацією HTML5 та стилізацією станів.

## Структура проєкту
lab01-html/ 
├── index.html
├── pages/
│ ├── about.html
│ └── contact.html
├── styles/
│ ├── style.css (основні стилі)
│ └── responsive.css (адаптивні медіа-запити)
├── js/
│ └── app.js (бонусний скрипт – кнопка зі зміною тексту)
├── assets/
│ ├── img/ (не використовується, зображення з мережі)
│ └── media/ (можна додати локальні медіа)
├── screenshots/ (папка для скріншотів – створити самостійно)
│ ├── desktop.png
│ └── mobile.png
└── README.md

## Як запустити

1. Завантажте всі файли проєкту, зберігаючи структуру папок.
2. Відкрийте файл `index.html` у будь-якому сучасному браузері (Chrome, Firefox, Edge).
3. Для перевірки адаптивності скористайтеся інструментами розробника (F12) або зменште вікно браузера до ширини <768px.
4. Форма не надсилає дані на сервер (метод `GET` – дані з'являються в URL), це навчальний приклад.
